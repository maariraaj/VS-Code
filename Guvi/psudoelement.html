CSS Pseudo-elements: Double colons (::) are used for pseudo-elements. This distinguishes pseudo-elements from pseudo-classes that use single colon (:) in their notation.

For example, it can be used to: Style the first letter, or line, of an element. Insert content before, or after, the content of an element
