CSS Units(measurements)
  1. Absolute Unit: Non dependent on something. Ex:(px) px is the default.
  2. Relative Unit: Dependent on something. Ex: (em,rem,%,vh,rw,ch)

    How to choose units?
  • px: Pixels, absolute length.
  • em: Relative to font size.(default fornt size is 16px)
  • rem: Relative to root element font size
  • vw: 1% of the viewport width
  • vh: 1% of the viewport height
  • ch: width of "0" character
  • %: percentage, usually relative to the root value
    When to use these units

    For width/height
  • %: to be relative to parent
  • vw/vh: to be relative to viewpoint
  • ch: for paragraph widths
  • rem: for closer to absolute values
  • px: true absolute

    • For margin/padding
  • rem: for closer to absolute value
  • em: to scale with font size
  • px: for small values

    • For Borders/Shadows
  • px: for small values
  • rem/em: optional (doesn't always look great)

    • For font size
  • rem: is usually the best
  • em: to scale to parent size
  • px: last option
