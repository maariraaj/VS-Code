CSS- Cascading Style Sheets
  • Primary styling language of the web
  • CSS is used to describe the style and layout of HTML document
  • CASCADING: The order that stylesheets are used
    1. User-Agent stylesheets (Browser default)
    2. User stylesheets (user preferences saved in browser)
    3. Author stylesheets (contain the CSS code we write)(highest level of precedence)
