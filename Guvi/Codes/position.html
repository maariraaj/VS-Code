Static: Default. Elements are positioned to the normal flow of the document.
Relative: The element is positioned relative to its normal position in the document flow.
    Absolute:
  • The element is removed from the normal document flow and no space is created for the element in the page layout.
  • It is positioned relative to its closest positioned ancestor (ancestor position value other than 'static')
    Fixed:
  • The element is removed from the normal document flow and no space is created for the element in the page layout.
  • It is positioned relative to the viewport, it will stay in the same place even if the page is scrolled.
    Sticky:
  • It is a hybrid of 'relative' and 'fixed'. The element is treated as 'relative' positioned until it crosses a specified threshold at which point it is treated as 'fixed'.
